Design a world-class premium web application for "SIMATS Online Education Portal".

The website must NOT look like a traditional university website.

Instead, it should look like a modern SaaS dashboard developed in 2026 with elegant Glassmorphism, smooth animations, premium typography, responsive layout, and enterprise-level UI.

Overall Design Style

• Apple Glass UI
• Windows 11 Fluent Design
• Glassmorphism
• Soft shadows
• Frosted Glass
• Gradient Accents
• Premium Animations
• Rounded corners (18-24px)
• Blur Effects
• Modern Academic Branding

The website should feel like:

"Apple + Notion + Linear + Microsoft Fluent + Premium LMS"

Never use old Bootstrap-style cards.
